# .net-learning
首先必须明确的一点是，.net不是一种开发语言，而是一个支持多种开发语言（C#、F#、VB等）的通用平台


# C#
一门新式编程语言，C语言发展演化的新产物，支持目前流行的编程范式（OOP），拥有可靠的垃圾回收机制
和异常处理机制，类型安全的设计亦是它的一个优点，除此之外，它还采用统一的类型系统，更具特色的是，
它在版本控制方面的处理远超大多数的编程语言。
